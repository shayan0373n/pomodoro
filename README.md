# Continuous Pomodoro

A persistent, gamified Pomodoro timer that never stops.

**Live Demo:** [https://pomodoro-five-gamma.vercel.app/](https://pomodoro-five-gamma.vercel.app/)

## Features
- **Continuous Timer:** The timer keeps running even if you close the tab (synced via localStorage).
- **Gamified Rewards:** Earn "Rest Minutes" for every completed Pomodoro.
- **Gift Store:** Spend your earned rest minutes when you need a break.
- **Trivia:** Learn interesting facts while you rest.
- **Audio & Notifications:** Get alerted when your focus or rest session ends.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
